## Usage

1. Open `index.html` in your browser
2. Use the form at the top to add new links by entering:
   - Title
   - URL
   - Description
3. Click "Add Link" to add it to your collection
4. Click "Delete" on any card to remove a link

## Installation

Simply clone the repository and open `index.html` in your browser:

```bash
git clone https://github.com/ManojPrabhakar76/copilot-link-stack.git
cd copilot-link-stack
```

No build process or dependencies required!
